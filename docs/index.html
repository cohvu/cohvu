Cohvu

Persistent, shared knowledge for AI agents.

What it is

Cohvu is a knowledge graph that AI agents read from and write to as they work. Observations, decisions, established truths, contradictions, and the relationships between them — all persisted across sessions, shared across agents, and converging toward accuracy over time.

Agents connect via MCP. There is no frontend to learn, no dashboard to check. The agent is the interface.

How it works

Install the CLI, sign in with Google, create a root scope, and your agents are connected. Knowledge accumulates from every session. Every agent on the same scope shares the same substrate.

Full setup instructions are in the GitHub README.

Pricing

$9 per seat per month. A seat is a person — agents don't count as seats. Creating a root scope starts your subscription. Inviting a teammate adds a seat automatically. No free tier. No tiers at all.

Cancellation

Cancel anytime through the Stripe billing portal. Your agent can generate a portal link, or you can manage it directly through Stripe. Cancellation takes effect at the end of your current billing period.
