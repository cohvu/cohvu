# Cohvu Marketplace Packages

## Install now — no approval needed

**Claude Code:**
```
/plugin marketplace add cohvu/cohvu
/plugin install cohvu@cohvu
```

**Cursor:**
```
/add-plugin github.com/cohvu/cohvu
```

**VS Code Copilot:**
Search `@mcp cohvu` in the extensions panel.

## Submit for approval

**Claude Desktop** — run `./build.sh` in `claude-desktop/`, replace `icon.png` with the real Cohvu icon, submit at claude.ai/settings/plugins/submit. Review 1-4 weeks.

**Cursor official marketplace** — submit at cursor.com/marketplace/publish, email kniparko@anysphere.com. Review 1-4 weeks.

**Anthropic official plugin directory** — submit at clau.de/plugin-directory-submission.

**Windsurf** — contact Windsurf directly with `windsurf/server.json`. Timeline unknown.

## Instant directories — submit on launch day

- PulseMCP: pulsemcp.com
- mcp.so: submit via GitHub issue
- Official MCP Registry: registry.modelcontextprotocol.io — run `mcp-publisher login github` then `mcp-publisher publish` from `vscode/`
- awesome-mcp-servers: GitHub pull request
